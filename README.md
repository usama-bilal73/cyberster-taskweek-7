# Web Security: SQL Injection Exploration & Mitigation

## Overview
This repository contains a comprehensive technical report on **Web Application Foundations and SQL Injection Mastery**. The project demonstrates the full lifecycle of a cyberattack, from environment setup to database compromise, emphasizing defensive secure coding practices.

## Lab Environment
*   **Target:** DVWA (Damn Vulnerable Web Application)
*   **Stack:** XAMPP (Apache & MariaDB)
*   **Tools:** Kali Linux, Burp Suite, SQLMap, CrackStation

## Key Technical Activities
*   **Manual Exploitation:** Logic bypass (Auth Bypass) and UNION-based data extraction.
*   **Advanced Testing:** Implementation of Boolean-based and Time-based (SLEEP commands) injections.
*   **Automation:** Full database dumping and password hash cracking using SQLMap.
*   **Defensive Analysis:** Development of mitigation strategies including WAF deployment and Prepared Statements.
